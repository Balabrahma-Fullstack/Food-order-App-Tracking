# 🍽️ FoodApp - Online Food Ordering Platform

## 📌 Project Overview

**FoodApp** is a full-stack MERN-based online food ordering platform developed by **Bala Brahma Chari**. The application enables customers to discover restaurants, browse menus, place food orders, and securely complete payments online. Restaurant owners can manage their restaurants, menus, customer orders, and track order statuses through a dedicated dashboard.

The platform focuses on delivering a seamless food ordering experience with secure authentication, real-time order management, and scalable cloud-based infrastructure.

## 🚀 Key Features

### Customer Features

* User registration and secure authentication using Auth0.
* Search restaurants by city, cuisine, or restaurant name.
* Browse restaurant menus and food items.
* Add items to cart and manage orders.
* Secure online payment using Stripe Checkout.
* View order history and track order status.
* Update user profile and delivery details.

### Restaurant Owner Features

* Create and manage restaurant profiles.
* Upload restaurant images using Cloudinary.
* Add, update, and manage menu items.
* View incoming customer orders.
* Update order statuses (Placed, Paid, In Progress, Out for Delivery, Delivered).
* Manage restaurant details and delivery settings.
